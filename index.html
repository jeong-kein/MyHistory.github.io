Chatwork Chatwork를 활용하여 프로젝트의 시각화를 통해협업,의사결정 등의 커뮤니케이션을 할 수 있습니다.
Slack Slack을 활용하여 원활한 커뮤니케이션을 할 수 있습니다.
Mattermost Mattermost을 활용하여 원활한 커뮤니케이션을 할 수 있습니다.
Teams Mattermost을 활용하여 원활한 커뮤니케이션을 할 수 있습니다.
GitHub GitLab를 활용하여 Branch 관리및 전체 프로젝트 관리를 할 수 있습니다.
AWS Google Analytics를 활용하여 데이터의 수집및 관리를 할 수 있습니다.
Ruby Ruby를 기반으로 Spring이나 Netty등을 활용하여 개발을 할 수 있습니다. 또한 JQuery나 JavaScript를 활용한 화면단 부터 백앤드 까지 Full Stack개발을 담당하고 있습니다.
PHP PHP를 기반으로Linux를 비롯한 여러 환경에서 웹 사이트를 개발 할 수 있습니다. 현재는 설계과정부터 업무 수행이 가능한 수준을 목표로 노력 중입니다.
Python Python를 기반으로Linux를 비롯한 여러 환경에서 웹 사이트를 개발 할 수 있습니다. 현재는 설계과정부터 업무 수행이 가능한 수준을 목표로 노력 중입니다.
JavaScript JavaScript, ECMAScript, Webpack 과 Babel등을 위주로 개발하며 모듈화를 통한 플러그인 개발 및 SDK도 가능합니다.
브라우저 환경의 JavaScript의 경우 브라우저 호환성을 중요시하여 IE 환경에서도 정상 작동하도록 개발하고 있으며 최신 문법인 ES6 또한 사용하고 있습니다
Vue Redux와 Slice로 상태 관리를 하며 Axios를 주로 사용하여 API 요청을 구현합니다.
대부분의 필요한 웹 컴포넌트를 능숙하게 개발할 수 있습니다(CRUD, Formik, Yup 등..) 또한 Node.js로부터 JWT를 발급받아 사용자 인증을 구현할 수 있습니다
MongoDB 관계형 데이터베이스를 설계하고 Mongoose를 사용하여 높은 생산성으로 API를 개발하고 있습니다
Camel case로 데이터 모델링 시 필드명을 정의하여 설계하며 데이터 ID 값은 MongoDB 내장 값인 _id를 증감 시켜 사용하고 있습니다
MySQL 관계형 데이터베이스를 설계하고 Django 나 SQL Developer를 사용하여 API를 개발하고 있습니다
Oracle 관계형 데이터베이스를 설계하고 SQL Developer를 사용하여 API를 개발하고 있습니다
PostgreSQL 관계형 데이터베이스를 설계하고 SQL Developer를 사용하여 API를 개발하고 있습니다
